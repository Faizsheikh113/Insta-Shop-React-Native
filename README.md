# Insta Shop - React Native Ecommerce App

A modern ecommerce mobile application built using React Native with clean UI, smooth navigation, scalable architecture, and responsive design.

Built with:

* React Native
* JavaScript
* Redux Toolkit
* React Navigation
* REST APIs
* Modern UI/UX

---

# Features

* Beautiful onboarding screens
* Authentication flow
* Home dashboard with categories
* Product listing
* Product detail screen
* Shopping cart
* Wishlist support
* Profile section
* Responsive UI
* Android support

# Getting Started

## Clone Repository

```bash
git clone <your-repository-url>
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Metro Server

```bash
npx react-native start
```

---

## Run Android App

```bash
npx react-native run-android
```
